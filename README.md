# Encore
Display what you're watching or listening to on your Discord profile.

> [!WARNING]
> Encore is under development and not ready to be used.

## Preview
<img src="Preview.png" alt="Preview" />

## Installation
1. Download the latest `dmg` from the [releases](https://github.com/getencore/Encore/releases)
2. Drag and drop the `app` into the `Applications` directory

## Compiling
### macOS
1. Install the dependencies:
    - `Xcode`: 14 or later
2. Clone the project: `git clone https://github.com/getencore/Encore.git && cd ./Encore/ && open ./App.xcworkspace/`
3. Build and run the project
